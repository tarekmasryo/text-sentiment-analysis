# Artifacts

This folder stores outputs produced by the notebook (generated locally or on Kaggle):

- `tfidf_vectorizer.joblib`
- `<best_model>_model.joblib`
- `model_summary.csv`
- optional plots / metrics exports

Notes:
- Files under `artifacts/` are **gitignored** by default (except this README and `.gitkeep`).
- Keep this folder clean; re-generate artifacts by re-running the notebook.
